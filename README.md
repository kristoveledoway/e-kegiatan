## Tentang Proyek ini

### Prerequisites

Dibangun dengan website lokalan .

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/kristoveledoway/e-kegiatan.git
   ```
2. Sesuaikan nama database
   ```sh
   import database manual di phpmyadmin
   ```
3. Buka via terminal/cmd/cli
   ```js
   php spark serve
   ```
